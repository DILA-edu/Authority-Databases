Description
===========

本佛經目錄規範資料製作之目的，乃以 CBETA 所收錄之佛典資料為基礎，重新統整相關經錄資料與近代研究成果，並整合其他三個規範資料庫，形成為一人、時、地、書目皆具的完整參考資料庫。
